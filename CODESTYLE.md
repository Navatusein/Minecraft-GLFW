## Стиль написания класов
```C++
class FooBar { // Фигурные скобки открываются в строке с объявление идентификатора класса
    public:
        FooBar();

        void Аoo(); // Название переменных из одного слова
        void Bar();

        void setmNumber // Сеттеры и геттеры называються первое слово с маленькой

    protected:
        void BarFoo(); // Название методов из нескольких слов

        int FooNumber // Название переменной-члена из двух слов

    private:
        void FooBar();

        int mNumber; // Название переменной-члена начинающейся с одной буквы
};


FooBar::FooBar(FooNumber) : FooNumber(FooNumber){}; // Премер установки значений переменным класса в конструкторе 

```
